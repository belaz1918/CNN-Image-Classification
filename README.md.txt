# CNN Image Classification

This Jupyter notebook implements a Convolutional Neural Network (CNN) for classifying images using PyTorch.

## Features
- CNN model with configurable architecture
- Training and evaluation on standard datasets (e.g., CIFAR-10, MNIST)
- Visualizations and accuracy tracking

## How to Use
Run this notebook in Google Colab or Jupyter Lab:

- [ ] Upload your dataset or use torchvision datasets
- [ ] Run training cells
- [ ] Evaluate the model on the test set
